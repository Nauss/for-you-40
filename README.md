# Birthday site prototype

Mini site statique mobile-first pour révéler un cadeau d'anniversaire.

## Lancer en local

Depuis ce dossier :

```bash
python3 -m http.server 8080
```

Puis ouvrir :

- http://localhost:8080
- ou depuis un autre appareil du même réseau : http://IP_DE_TA_MACHINE:8080

## Déploiement GitHub Pages

Comme c'est un site statique, il peut être publié tel quel sur GitHub Pages.

## Fichiers

- `index.html` — structure
- `styles.css` — design
- `script.js` — navigation entre écrans

## Prochaines itérations possibles

- personnaliser les textes
- ajouter musique via bouton d'activation
- ajouter QR code imprimable
- affiner les animations
- ajouter visuels / textures / illustrations discrètes
